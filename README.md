## Hướng dẫn Build và Nạp code

1.  Mở project bằng STM32CubeIDE.
2.  Nhấn `Build` (Ctrl + B).
3.  Kết nối mạch nạp ST-Link và nhấn `Run` (F11) để nạp chương trình.
